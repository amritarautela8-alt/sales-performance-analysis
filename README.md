# Sales Performance Analysis

A comprehensive project analyzing sales data to uncover trends, insights, and performance metrics using Python for Exploratory Data Analysis (EDA) and Power BI for interactive dashboards.

## Project Overview
This project focuses on analyzing sales data to identify key patterns, trends, and opportunities for business growth. Python was used for data cleaning, analysis, and visualization, and Power BI was used to build an interactive dashboard to present actionable insights.

## Technologies Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Power BI
- Jupyter Notebook
- Excel / CSV for datasets

## Features
- Cleaned and preprocessed raw sales data
- Explored data to find trends, seasonality, and top-performing products
- Visualized insights using bar charts, heatmaps, and line plots
- Built an interactive Power BI dashboard showing:
  - Total sales over time
  - Regional performance
  - Product category performance
  - Monthly and quarterly growth trends
- Highlighted actionable insights for business decision-making
